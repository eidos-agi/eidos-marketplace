# Changelog

All notable changes to Converge are recorded here.

This project follows the spirit of Keep a Changelog and uses semantic versions.

## [0.1.0] - 2026-05-24

### Added

- Contract-first Converge Codex skill.
- Starter target lattice template and JSON schema.
- Portable adapter row schema.
- JSON reference adapter for target lattices.
- Pytest/JUnit adapter that emits Converge rows.
- Row aggregator that validates, merges, scores, preserves conflicts, and emits repair rows.
- Eidos plugin manifest, playbook, and verification hook.
- Claude plugin manifest for marketplace distribution.
